// 如何添加文章

  1. 将 .md 或 .pdf 文件放入仓库的 articles/ 目录
  2. 编辑 data/articles.json，添加文章的元数据（标题、标签、关键词等）
  3. 推送到 GitHub，GitHub Actions 会自动部署到 GitHub Pages
  4. 也可点击上方 New Article 按钮，复制生成的 JSON 条目后手动添加
